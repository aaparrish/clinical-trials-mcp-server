Advanced Query

Condition: Metastatic non-small cell lung cancer (NSCLC), adenocarcinoma

Age/Sex: 68, Male

Location: Seattle, WA (willing to travel ~2 hours)

Progression: After 1st-line osimertinib; EGFR T790M negative, MET amplification detected

Medical: COPD (moderate), on apixaban for AFib, prior platinum doublet chemo, AST 1.5x ULN, no brain mets

Biomarkers: EGFR exon 19 del, MET amplification, PD-L1 5%

Constraints: Must allow prior EGFR-TKI/chemo, accept MET+ patients

Installation

Set up in minutes with Claude Desktop

1. Clone the Repository

git clone https://github.com/aaparrish/clinical-trials-mcp-server.git cd clinical-trials-mcp-server

2. Install Dependencies

pip install mcp httpx

3. Configure Claude Desktop

Add to your claude_desktop_config.json:

{ "mcpServers": { "clinical-trials": { "command": "python3", "args": ["/path/to/clinical-trials-mcp-server/clinical_trials_mcp_server.py"] } } }

4. Restart Claude Desktop

Quit and reopen Claude Desktop. You're ready to search clinical trials!
